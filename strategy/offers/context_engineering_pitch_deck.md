## Slide 4: The Solution
**Visual:** A clean, schematic diagram. An "Architect" icon placing a structured block called `CLAUDE.md` into a slot, instantly stabilizing the agent.
**Headline:** Enter Context Engineering
**Body Copy:**
**Context Engineering** is the discipline of designing systems that provide language models with maximally useful information at minimal token cost.
It is **not** Prompt Engineering 2.0.
*   Prompt Engineering = What you say to the model.
*   Context Engineering = What the model knows, sees, and remembers.

## Slide 5: The Framework (The Context Stack)
**Visual:** The "Context Stack" Hero Image again, but labeled with the 4 Layers.
**Headline:** The Context Stack
**Body Copy:**
We don't just "chat." We build infrastructure across four layers:
1.  **Immediate (Level 0):** The visible prompt. Optimization for signal density.
2.  **Session (Level 1):** Managing the flow of a single task.
3.  **Project (Level 2):** Persistent memory (e.g., `CLAUDE.md`). The "Instruction Manual" for your project.
4.  **Reference (Level 3):** On-demand libraries. "Sacks of knowledge" loaded only when needed.
