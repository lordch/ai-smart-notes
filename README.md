# AI Smart Notes

Baza wiedzy służąca rozwojowi praktyki konsultingowej w obszarze Human-AI Collaboration.

---

## Struktura

```
ai-smart-notes/
├── concepts/       # Moje przemyślenia, framework, oryginalne koncepty
├── literature/     # Notatki z tekstów, papers, podcastów
├── sources/        # Surowe materiały (transkrypty, artykuły)
├── strategy/       # Strategia i operacje biznesowe
│   ├── positioning/    # Narracje, messaging
│   ├── offers/         # Opisy usług
│   ├── clients/        # Notatki o klientach
│   └── plans/          # Plany, roadmapy
├── blog/           # Content do publikacji
└── tasks/          # Transient - foldery per zadanie/sesja
```

## Warstwy wiedzy

| Warstwa | Folder | Opis |
|---------|--------|------|
| **Źródła** | `sources/` | Surowy materiał — transkrypty, artykuły, bez edycji |
| **Literatura** | `literature/` | Moje notatki o cudzych myślach |
| **Koncepty** | `concepts/` | Moje oryginalne przemyślenia |
| **Strategia** | `strategy/` | Biznes — pozycjonowanie, oferty, klienci |
| **Blog** | `blog/` | Content gotowy/w trakcie do publikacji |
| **Tasks** | `tasks/` | Transient — foldery per zadanie |

---

## Przepływ wiedzy

```
sources/ → literature/ → concepts/ → strategy/offers/
                                  → blog/
```

---

## Tasks (praca transient)

Dla każdego większego zadania twórz folder w `tasks/`:
```
tasks/2024-01-15-mspark-prep/
├── notes.md
├── research.md
└── draft.md
```

Notatki mogą zostać "wypromowane" do trwałych warstw lub usunięte po zakończeniu.

---

## Konwencje nazewnictwa

- `Literature - {Autor lub Tytuł}.md` — notatki ze źródeł
- `Strategy - {Temat}.md` — dokumenty strategiczne
- `Blog - {Tytuł}.md` — posty blogowe
