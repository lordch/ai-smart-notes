# Tasks

Foldery transient — jeden folder per zadanie/sesja.

---

## Jak używać

1. Utwórz folder z datą i krótkim opisem:
   ```
   tasks/2024-01-15-mspark-prep/
   ```

2. Wewnątrz twórz notatki robocze:
   ```
   tasks/2024-01-15-mspark-prep/
   ├── notes.md        # Luźne notatki
   ├── research.md     # Research
   └── draft.md        # Drafty
   ```
