### Inspiracje z Litterfy AI (2026-01-22)

Źródło: transkrypt YouTube o narzędziu Litterfy AI do literature review.

**Co warto zaadaptować:**

1. **Multi-source search** — przeszukiwanie PubMed + Google Scholar + Semantic Scholar jednocześnie → mniej "czy czegoś nie pominęłam?"

2. **Sortowanie przed screeningiem** — ranking po: relevance, roku, liczbie cytowań. Agent może pokazać "top 30 najbardziej cytowanych z ostatnich 5 lat dotyczących twojego PICO"

3. **Batch export metadanych** — eksport listy paperów do importu do Zotero zamiast ręcznego dodawania

4. **Outline-first synthesis** — najpierw edytowalny outline (grupowanie paperów tematycznie), potem dopiero generowanie prozy

5. **Template-driven review** — gotowy szablon "Pragmatic lit review dla badania retrospektywnego"

**Gdzie agent jest lepszy niż Litterfy:**

| Litterfy | Agent |
|----------|-------|
| Na podstawie abstraktów | Czyta pełne PDF-y |
| Nie zna kontekstu badania | Zna hipotezę, PICO, cel |
| Jednorazowy output | Iteracyjna rozmowa |
| Brak ekstrakcji danych | Ustrukturyzowana ekstrakcja |
| Brak snowballing | Może analizować cytowania |

**Konkretne pomysły:**

- "Screening assistant" — agent + lista abstraktów + kryteria → ranking z uzasadnieniami
- "Paper extraction template" — standardowy formularz per paper
- "Coverage check" — agent sprawdza czy lista pokrywa główne wątki tematu
